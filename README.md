
Download oh-my-zsh
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

Download Brew files
```
brew bundle
```

To delete all the exising configs
```bash
Make clean
```

To update, `cd` into your local `dotfiles` repository and then:
```bash
Make sync
```
